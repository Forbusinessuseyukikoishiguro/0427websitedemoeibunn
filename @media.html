@media クエリとは？

@media クエリは、CSS3の機能で、デバイスの特性（画面サイズなど）に基づいてスタイルを適用することができます。これにより、さまざまなデバイスに最適化されたレスポンシブウェブデザインを実現できます。

問題点

画面サイズが小さくなると、コンテンツとサイドバーが並列に表示されるとスペースが狭くなり、読みにくくなります。また、ナビゲーションリンクも狭くなり、タップしづらくなります。

解決策

@media クエリを使用して、画面サイズに応じてレイアウトを変更します：

/* タブレット用のスタイル */ @media screen and (max-width: 768px) { .main-content { flex-direction: column; } .sidebar { margin-left: 0; margin-top: 20px; } }

これにより、タブレットサイズでは、サイドバーがメインコンテンツの下に移動し、十分な幅を確保できます。

デスクトップ

画面幅が広いデバイス向けのデザイン。通常、複数カラムのレイアウトが効果的です。

タブレット

中程度の画面幅のデバイス。デスクトップほど複雑なレイアウトは避けるべきです。

モバイル

小さな画面幅のデバイス。シンプルな単一カラムのレイアウトが最適です。

@media クエリ例

画面幅に基づくメディアクエリ：

@media screen and (max-width: 768px) { ... }

印刷用のスタイル：

@media print { ... }

向きに基づくメディアクエリ：

@media (orientation: portrait) { ... }

解像度に基づくメディアクエリ：

@media (-webkit-min-device-pixel-ratio: 2) { ... }
